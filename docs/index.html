Core tools

These scripts live under .\tools\ and are designed to be idempotent, safe to re-run, and friendly for future contributors.

Content

tools\Add-Post.ps1

Adds a new post skeleton in posts\YEAR\MM\slug\index.html, wired to the unified layout and card grid.

pwsh .\tools\Add-Post.ps1
Automation

tools\find-post.ps1

Quick search by slug; opens the exact post folder so you never edit the wrong copy.

pwsh .\tools\find-post.ps1 new-devops-insights -Open
Quality

tools\check-slugs.ps1

Scans all posts and projects to make sure slugs are unique and match the [a-z0-9-] pattern.

pwsh .\tools\check-slugs.ps1
Quality

tools\check-missing-tags.ps1

Finds posts missing <meta name="tags"> and suggests tags based on existing content.

pwsh .\tools\check-missing-tags.ps1
Quality

tools\fix-missing-tags.ps1

Applies the suggested tags, creates backups, and keeps metadata consistent.

pwsh .\tools\fix-missing-tags.ps1
Safety

tools\Safe-Rebase.ps1

Guarded rebase: checks for a clean working tree, creates a backup branch, then rebases safely.

pwsh .\tools\Safe-Rebase.ps1
